# 🎬 Editkaro Portfolio Website

A modern and responsive portfolio website for **Editkaro.in**, a Social Media Marketing and Video Editing Agency. Built using **HTML, CSS, and JavaScript** with a focus on clean design, smooth animations, and an engaging user experience.

## ✨ Features

- Responsive design
- Interactive portfolio gallery
- Category-based filtering
- Smooth animations
- Modern UI/UX
- Contact section
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

```
│── script.js
│── assets/
```
